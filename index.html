Lossless and Lossy Compression

Table showing the features of lossless and lossy compression
Features of Lossless Compression Features of Lossy Compression
All the original data can be recovered when the file is uncompressed Some of the data is permanently removed during compression and cannot be recovered
Used with text as all data must be recoverable Provide more compression than lossless techniques
Example techniques run-length encoding, dictionary encoding Popular in situations where the removal of data is not noticeable as in the case of images and audio
Example file formats .zip, .png Example file formats .jpg, .gif, .mp3
Not normally applied to video as higher compression rates are needed to make files smaller
